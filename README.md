# Fiszki rekrutacyjne (PWA)

Prosta aplikacja typu fiszki (200 pytan i odpowiedzi) do przygotowania na rozmowy:
**PHP, Symfony, Doctrine/SQL, API/HTTP, bezpieczenstwo, testy, React, TypeScript, architektura.**

- Tapnij karte = pokaz/ukryj odpowiedz
- Swipe lewo/prawo lub strzalki = kolejne pytanie
- Filtr kategorii, wyszukiwarka, losowanie, zapis postepu
- Dziala offline (PWA, service worker)

## Wlaczenie GitHub Pages
1. **Settings -> Pages**
2. **Source:** Deploy from a branch
3. **Branch:** `main` + folder `/ (root)` -> **Save**
4. Strona bedzie pod: `https://irekmi.github.io/pechafiszki/`

## Instalacja na iPhone (jak aplikacja)
1. Otworz link w **Safari**
2. **Udostepnij -> Dodaj do ekranu poczatkowego -> Dodaj**
3. Otworzy sie pelnoekranowo, jak natywna apka (tez offline)
